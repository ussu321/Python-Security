## 🌐 Overview

**Ethical Hacking Python** is a next-generation cybersecurity intelligence platform designed for **educational**, **defensive**, and **ethical hacking** purposes. Built as a futuristic cyberpunk SOC dashboard, it combines:

- 🤖 **AI-inspired security analysis** with contextual insights
- 🔍 **Advanced network scanning** with topology visualization
- 🌐 **OSINT domain intelligence** (WHOIS, DNS, SSL, subdomains)
- 🔐 **Password strength analysis** with entropy scoring
- 🧪 **Cryptographic hashing laboratory** (MD5, SHA family, BLAKE2)
- 📁 **File security analyzer** with upload/download systems
- ⚠️ **Vulnerability assessment simulation** with CVSS scoring
- 📊 **Packet analytics** and system monitoring
- 🧠 **Cyber threat intelligence** with visualizations
- 📈 **Advanced analytics engine** (radar, heatmaps, node graphs)

> ⚡ **Mission:** Empower cybersecurity students, professionals, and enthusiasts with a safe, visual, and interactive defensive toolkit.

## 🚀 Installation

### Prerequisites
- **Python 3.11+** (3.12 recommended)
- **pip** package manager
- **Git** (for cloning)

### Linux / Kali Linux / Ubuntu / Debian

```bash
# Clone the repository
git clone https://github.com/issu321/Ethical-Hacking-Python.git
cd Ethical-Hacking-Python

# Run the automated installer
bash install.sh
```

The installer will:
1. ✅ Check Python version
2. ✅ Create a virtual environment (`venv/`)
3. ✅ Install all dependencies from `requirements.txt`
4. ✅ Launch Streamlit automatically

**Manual installation:**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**System dependencies (if scapy fails):**
```bash
sudo apt-get update
sudo apt-get install -y libpcap-dev
```

### Windows 10 / 11

```powershell
# Clone the repository
git clone https://github.com/issu321/Ethical-Hacking-Python.git
cd Ethical-Hacking-Python

# Run the automated installer
.\install.bat
```

**Manual installation:**
```powershell
python -m venv venv
.env\Scriptsctivate
pip install -r requirements.txt
streamlit run app.py
```

**Note for Windows:**
- Install [Npcap](https://npcap.com/) for full packet capture features
- Run as Administrator for raw socket operations
- The app gracefully falls back to simulated data without admin rights

## 🛠️ Tool Explanations

### Advanced Network Scanner
Uses Python's `socket` library for safe TCP connect scanning. Multi-threaded for speed. Detects open ports, attempts safe banner grabbing, and classifies severity. Results are visualized as:
- Interactive data tables
- Port status heatmaps
- Network topology node graphs
- Risk score gauges

### Domain Intelligence
Integrates `python-whois`, `dnspython`, and `ssl` modules to gather:
- Registration details and domain age
- DNS records across multiple types
- SSL certificate validity and cipher info
- Simulated reputation scoring for threat visualization

### Password Analyzer
Implements Shannon entropy calculation and pattern recognition:
- Checks against common password dictionaries
- Detects sequential characters and keyboard patterns
- Identifies date/year patterns
- Provides AI-generated hardening recommendations

### Hashing Laboratory
Supports 8 hash algorithms using Python's `hashlib`:
- MD5, SHA1, SHA256, SHA512, SHA3-256, SHA3-512, BLAKE2b, BLAKE2s
- Text and file input
- Integrity verification with hash comparison

### File Security Analyzer
Performs static analysis on uploaded files:
- Magic bytes identification
- MIME type guessing
- Entropy analysis (packed/encrypted detection)
- Dangerous extension warnings
- Comprehensive hash generation

### Vulnerability Assessment
SAFE simulation engine that maps open ports to known vulnerability patterns:
- Simulated CVSS scoring
- Severity classification (Critical/High/Medium/Low)
- Defensive recommendations
- Risk heatmaps

## 📤 Upload & Download Guide

### Uploading Files
1. Navigate to **📂 Upload Center**
2. Drag and drop or select files (TXT, CSV, JSON, LOG, PCAP)
3. Files are automatically parsed:
   - **CSV** → DataFrame + auto bar charts
   - **JSON** → Structured tree view
   - **TXT/LOG** → Terminal-style display
4. Upload history is tracked in session

### Downloading Reports
1. Navigate to **📥 Download Center**
2. View all generated reports with timestamps
3. Export individual reports as:
   - 📄 **JSON** — Full structured data
   - 📊 **CSV** — Tabular data export
   - 📝 **TXT** — Formatted security report
4. Use **Bulk Export** to download all reports as a single JSON file
